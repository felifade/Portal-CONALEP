Hora 1
Un CSS para todas las páginas
Clave Un solo archivo style.css → diseño consistente en las 5 páginas del sitio. Si cambias un color, cambia en todas al mismo tiempo.
index.html & paginas/*.html 
/* index.html — en la raíz */
<link rel="stylesheet" href="css/style.css">

/* paginas/*.html — hay que subir un nivel */
<link rel="stylesheet" href="../css/style.css">
💡 El ../ sube un nivel en la carpeta — exactamente igual que la ruta de las imágenes desde paginas/.
Hora 2
Imágenes con estilo y efecto hover
Clave max-width limita el tamaño · border-radius redondea · box-shadow da profundidad · transition suaviza el efecto hover.
css/style.css 
img {
  max-width:   280px;
  display:     block;
  border-radius: 8px;
  box-shadow:  0 4px 16px rgba(0, 0, 0, 0.6);
  transition:  transform 0.3s ease;
}

img:hover {
  transform: scale(1.05);   /* zoom suave */
}
✨ El hover aplica a todas las imágenes del sitio sin tocar el HTML. Con border-radius: 50% se hacen círculos.
Hora 3
Flexbox: galería que se organiza sola
Clave display: flex se aplica al contenedor, no a las imágenes. flex-wrap: wrap permite que pasen a la siguiente línea automáticamente.
css/style.css 
main {
  max-width: 960px;
  margin:    0 auto;    /* centrar */
  padding:   24px 20px;
}

.galeria {
  display:         flex;
  flex-wrap:       wrap;
  gap:             16px;
  justify-content: center;
}
// Cómo se ven las imágenes con Flexbox + flex-wrap: wrap
div.galeria { display: flex; flex-wrap: wrap; }
